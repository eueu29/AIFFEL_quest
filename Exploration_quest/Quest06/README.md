- 생성되는 이미지 퀄리티가 좋아서 일단 신기했으나 생각보다 프롬프트대로 말을 듣게 하려면 명령어를 잘 입력해주는 방법을 먼저 터득해야 할 듯 하다.

  - 실험을 위해 윤곽선 검출은 배경이 될 이미지도 넣어보았으나 별 차이가 없었다
